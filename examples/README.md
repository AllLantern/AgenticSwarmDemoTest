# Demo Project Examples

These are starter templates for the live demo. Each can be expanded based on audience suggestions!

## Available Templates

### 1. TypeScript CLI (`typescript-cli/`)
A command-line tool built with TypeScript.

```bash
cd typescript-cli
npm install
npm run dev
```

**Good for demos like:**
- File organizer
- Password generator
- JSON/CSV converter
- Todo list manager

### 2. Go Tool (`go-tool/`)
A command-line tool built with Go.

```bash
cd go-tool
go run .
```

**Good for demos like:**
- System utilities
- Network tools
- File processors
- Build tools

### 3. Web API (`web-api/`)
A REST API built with Node.js.

```bash
cd web-api
npm install
npm run dev
```

**Good for demos like:**
- CRUD API
- Authentication service
- Webhook handler
- Data aggregator

## During the Demo

1. **Audience suggests** what to build
2. **Planner** breaks it into steps
3. **TDD Guide** writes failing tests
4. **We implement** to make tests pass
5. **Reviewers** check quality and security
6. **Ship it!**

## Quick Commands

```bash
# TypeScript CLI
cd typescript-cli && npm test

# Go Tool
cd go-tool && go test ./...

# Web API
cd web-api && npm test
```
